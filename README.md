# AI SEO Pilot

Optimize your WordPress site for AI search engines (ChatGPT, Perplexity, Claude, Gemini) through GEO/AEO best practices.

## Features

- **llms.txt Generation** — Auto-generate or manually craft your site's `llms.txt` file for AI crawlers
- **Schema.org JSON-LD** — AI-optimized structured data with speakable markup and automatic content type detection
- **Content Analysis** — Real-time scoring (0–100) with 10 AI-readiness checks and actionable suggestions
- **AI Bot Tracking** — Monitor visits from GPTBot, ClaudeBot, PerplexityBot, and 7 more AI crawlers
- **AI-Optimized Sitemap** — Custom XML sitemap with AI-specific metadata (content type, reading level, word count)
- **robots.txt Enhancement** — Automatic `Allow` directives for AI bots

## Requirements

- WordPress 6.0+
- PHP 7.4+

## Installation

1. Upload the `ai-seo-pilot` folder to `/wp-content/plugins/`
2. Activate the plugin through the **Plugins** menu
3. Go to **AI SEO Pilot** in the admin menu to configure

## FAQ

### Does this replace Yoast or Rank Math?

No. AI SEO Pilot complements traditional SEO plugins by focusing specifically on AI search engine optimization (GEO/AEO).

### What AI bots does it track?

GPTBot, ChatGPT-User, Claude-Web, ClaudeBot, PerplexityBot, Google-Extended, Amazonbot, Bytespider, cohere-ai, and YouBot.

### Is my data safe?

IP addresses are anonymized before storage. Bot visit data is stored locally in your WordPress database.
